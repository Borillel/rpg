  # Proposito

  Este projeto foi criado utilazando React, Vite, Tailwind CSS e React Router
  
  Esse projeto teve como intenção criar um site de rpg para eu jogar com meus amigos
  
  Como padrão de design eu escolhi o Decorator no sistema de compras de itens 

  ## Rodar o codigo

  Usar `npm i` instala todas as depedencias.

  Usar `npm run dev` ativa o sever para rodar o site.
